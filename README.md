# MedCare 🏥

A full-stack healthcare web application with user authentication, doctor management, appointment booking, and an admin panel.

## 🚀 Features
- 🔐 Login & Register (Spring Boot + MongoDB)
- 👨‍⚕️ Doctors listing
- 📅 Book appointments
- 👑 Admin panel (Add/Delete doctors)
- 🤖 AI Chatbot (health assistant)
- 🧭 Role-based access (admin vs user)

## 🧰 Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Spring Boot (Java 17)
- Database: MongoDB
